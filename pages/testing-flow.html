使用說明

  1. 選擇測試頁面 -- 上方下拉選單選 Home 或 Events
  2. 逐步測試 -- 點擊每個 Step 展開操作指引，按照步驟執行
  3. 標記結果 -- 每步完成後點 Pass / Fail / N/A
  4. 貼截圖 -- 發現問題時 Win+Shift+S 截圖，然後 Ctrl+V 貼到該步驟的截圖區
  5. 寫備註 -- 在備註欄描述問題（axe 報告重點、觀察結果等）
  6. 開 Issue -- Fail 的項目會出現「Open GitHub Issue」按鈕，點擊自動建立 issue
  7. 匯出報告 -- 隨時可點頂部「Export HTML Report」下載報告（含截圖）。未完成項目會提醒但不阻擋匯出

測試結果自動儲存在你的瀏覽器中。換電腦或清除瀏覽器資料會遺失。完成後請務必匯出報告。

P1 必測 Phase 1 必須測試（21 步）  條件式 看功能決定，沒有就標 N/A（5 步）  Phase 2+ 延後到有該功能時再測（4 步）

Phase 1 預估時間：每頁約 60-90 分鐘（首次測試）。Phase 2+ 項目目前可跳過。
