Phase 1 範圍（MOU 約定）

  • 首頁（Home）+ 活動頁（Events）
  • No-code 方案（Elementor / Divi / Oxygen）
  • WCAG 2.2 AA 無障礙合規
  • 中文為主，英文頁面留待 Phase 2
  • 回應式設計（桌機、平板、手機）
  • 每個里程碑最多 2 次修改

URL 結構建議

  • apac-aia.org/ → 首頁（中文）
  • apac-aia.org/events/ → 活動列表
  • apac-aia.org/events/[slug]/ → 活動詳情
  • apac-aia.org/en/ → 英文首頁（Phase 2）
  • apac-aia.org/en/events/ → 英文活動
  • 統一使用 /en/ 前綴，解決目前 /en/ vs /eng/ 混亂

無障礙要求（全站適用）

  • 所有圖片必須提供 alt text
  • 標題層級正確巢狀（H1 → H2 → H3）
  • 文字對比度通過 WCAG AA（見 Style Guide）
  • 鍵盤可完整導航
  • 表單欄位須有 label
  • 影音內容須有字幕 / 逐字稿

已備妥的設計資源

  • Brand Style Guide v1.0（HTML 格式）
  • SVG Logo × 5 種變體
  • 品牌色碼 × 5 色（WCAG 驗證完畢）
  • 字型：Noto Serif TC / Noto Sans TC / Inter
  • 深色模式色彩對應規則（Phase 2+ 備用）
  • 字型選擇決策紀錄
